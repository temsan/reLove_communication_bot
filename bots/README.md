# Pranayama Bot

Бот для автоматической обработки и публикации видео практик пранаямы.

## Функции

1. **Прием Zoom-ссылок** - админ отправляет ссылку на запись
2. **Скачивание видео** - автоматическое скачивание через Selenium
3. **Проверка содержания** - транскрибация и проверка, что это практика пранаямы
4. **Обработка видео**:
   - Кроп в вертикальный формат (9:16)
   - Очистка звука от шумов
   - Наложение ватермарка
5. **Генерация постов** - 3 варианта поста с учетом астрокалендаря
6. **Публикация** - выбор варианта и публикация в канал

## Установка

```bash
# Установи зависимости
pip install aiogram openai-whisper

# Создай .env файл
echo "PRANAYAMA_BOT_TOKEN=your_bot_token" > .env

# Запусти бота
python bots/pranayama_bot.py
```

## Использование

1. Запусти бота: `/start`
2. Отправь Zoom-ссылку или видео файл
3. Дождись обработки
4. Выбери вариант поста
5. Бот опубликует в канал

## Конфигурация

- `ADMIN_ID` - ID администратора (2366957431)
- `CHANNEL_ID` - ID канала (@reloverituals)
- `BOT_TOKEN` - токен бота
